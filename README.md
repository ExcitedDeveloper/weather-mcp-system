# Weather MCP System

A comprehensive weather information system consisting of:

- **MCP Server**: TypeScript-based server providing weather data via Open-Meteo API
- **React Client**: User-friendly web interface for weather queries
- **Testing Suite**: Comprehensive automated testing
- **Production Deployment**: Railway (server) + Vercel (client)
