# Weather MCP System

A comprehensive weather information system consisting of:

- **MCP Server**: TypeScript-based server providing weather data via Open-Meteo API
- **React Client**: User-friendly web interface for weather queries
- **Testing Suite**: Comprehensive automated testing
- **Production Deployment**: Railway (server) + Vercel (client)

## Project Structure

weather-mcp-system/
├── server/ # MCP server code
├── client/ # React application  
├── docs/ # Documentation
└── .github/ # GitHub workflows and templates

## Getting Started

### Server Development

cd server
npm install
npm run dev

## Features

- 🌤️ **Current Weather**: Real-time conditions for any location
- 📅 **Weather Forecasts**: 3-day outlook with detailed information
- 🚨 **Weather Alerts**: Active warnings and emergency notifications
- 🤖 **Smart Recommendations**: Clothing and activity advice
- 🔍 **Location Search**: Global geocoding and location resolution
